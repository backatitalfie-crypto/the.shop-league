# 🛒 Shop League

Welcome to the **official Shop League website**.

The Shop League is a fictional football league created and managed by me. Originally launched through **TikTok** and **Discord**, the project has expanded into this website to provide a central hub for fixtures, results, league tables, news, and club information.

This website has been built using **Adobe Dreamweaver**, **HTML**, **CSS**, and **JavaScript**.
